# Backend Development
># 📌 Overview

The Backend Development portion of the web development training at Humber College focuses on server‑side programming skills — the part of web applications that runs behind the scenes to handle data, logic, and integration with databases. At Humber’s North Campus, this is taught through specific courses such as Back‑End Web Development 1 (HTTP 5125), where students are introduced to server‑side programming using the C# language and learn techniques for creating data‑driven websites that pull and manage data from external sources.

># 🧠 Continued Learning and Skills

Following the introductory course, learners take Back‑End Web Development 2 (HTTP 5226), which builds on those fundamentals by having students work with a full‑stack coding framework — meaning they connect backend logic with the broader web application workflow and deepen their understanding of how all parts of a web app integrate. Together, these courses teach practical backend skills valuable for roles like backend or full‑stack developer by simulating real‑world development scenarios and reinforcing server‑side logic, data handling, and application workflow.
